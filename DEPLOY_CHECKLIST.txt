HOW TO UPDATE FILES IN GITHUB:
  1. Go to your GitHub repo
  2. Click on a file (e.g. main.py)
  3. Click the pencil icon (Edit)
  4. Select ALL content, DELETE it
  5. Paste the new file content
  6. Click "Commit changes"
  7. Repeat for each file
  8. For NEW files (Dockerfile, app.html, login.html):
     - Click "Add file" > "Create new file"
     - Type the filename exactly
     - Paste content
     - Commit

STEP 2: Redeploy on Render
  1. Go to dashboard.render.com
  2. Click your service
  3. Click "Manual Deploy" 
  4. Click "Clear build cache & deploy"
